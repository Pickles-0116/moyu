# 掌柜偷闲录

> 古风经营摸鱼小游戏 · 浏览器即开即玩 · 纯前端单机

---

## 项目概览

| 项 | 值 |
|---|-----|
| 技术栈 | 纯前端 HTML/CSS/JS，无框架无后端 |
| 数据存储 | `localStorage`（键名 `zhanggui_save`） |
| 核心循环 | 出发→行进动画→随机事件→A/B选择→反馈（15-30秒） |
| 四维资源 | 银两🪙 / 声望👑 / 体力❤️ / 商途值🗺️ |
| 老板键 | `ESC` — 一键切换为 Excel/记事本/代码编辑器/邮件客户端伪装 |

---

## 文件结构

```
src/
├── index.html          # 入口页面
├── css/
│   └── style.css       # 古风UI样式（宣纸/墨色/朱红/金配色）
└── js/
    ├── data.js         # 数据层：配置常量、事件池、道具、成就、传闻
    ├── game.js         # 引擎层：GameEngine — 资源/精力/循环/店铺/离线/存档
    └── ui.js           # 交互层：GameUI — 渲染/动画/飘字/教程/模态面板
```

架构模式：事件驱动 — `GameEngine emit()` → `GameUI on()`，松耦合。

## 核心系统

### 四维资源

| 资源 | 初始值 | 上限 | 说明 |
|------|--------|------|------|
| 银两 🪙 | 100 | 无 | 核心经济，买卖道具用 |
| 声望 👑 | 0 | 无 | 长线投资，解锁背包/路段/盐场门槛 |
| 体力 ❤️ | 100 | 100 | 生存约束，归零强制休整 |
| 商途值 🗺️ | 0 | 150软上限 | 进度可视化，解锁店铺/路段 |

### 精力系统

- 初始 20 点，每 5 分钟恢复 1 点
- 每次出发消耗 1 点
- 体力独立于精力（可无限行走但体力有限）

### 离线收益

- 基础：35% 店铺产出，上限 4 小时
- Prestige 后：50% 产出，上限 6 小时
- 活跃度衰减：操作后 15min 全效 → 70%（30min） → 50%（60min）
- 商机闪现：在线独享 5% 概率触发，30 秒产出翻倍

### 自动保存

- 资源变更触发 + 5 秒定时双重保障
- 支持导出/导入存档 JSON

## 店铺系统（6种）

| 店铺 | 图标 | 基础产出/10s | 解锁条件 | 特殊效果 |
|------|------|-------------|----------|----------|
| 茶馆 | 🍵 | 10 | 默认解锁 | — |
| 布庄 | 🧵 | 25 | 银两≥500 | — |
| 粮铺 | 🌾 | 60 | 银两≥2000 | — |
| 药铺 | 🏥 | 80 | 商途值≥200 | 每5级体力上限+10 |
| 酒楼 | 🍶 | 150 | 商途值≥300 | 每5级精力上限+1 |
| 镖局 | 🛡️ | 200 | 商途值≥800 | 每5级离线比例+5% |

### 产出公式

```
实际产出 = baseOutput × milestoneMultiplier + baseOutput × 0.02 × (level - 1)
```

里程碑倍率：Lv.10(×2) / Lv.25(×3) / Lv.50(×5) / Lv.100(×10)

### 店铺升级

```
升级费用 = upgradeBaseCost × upgradeCostMultiplier ^ level
```

各店升级倍率：茶馆 1.5 → 布庄 1.6 → 粮铺 1.7 → 药铺 1.7 → 酒楼 1.8 → 镖局 1.9

### 店铺变卖

- 首次回收 40% 累计投入，同一店铺再次变卖递减 10%
- 最终回收率下限 10%，冷却 24 小时

## 事件系统（30个）

事件分为 **12个类别**，触发时从事件池中随机抽取一个。

### 事件分类与数量

| 类别 | 数量 | 说明 |
|------|------|------|
| trade（交易） | 5 | 茶商/药商/小贩/乞丐/同行竞价 |
| gamble（赌博） | 4 | 古寺问签/赌坊/古董摊/斗茶 |
| risk（风险） | 3 | 山贼拦路/官差盘查/小贼扒窃 |
| cooperation（合作） | 2 | 同行邀约/商会招揽 |
| investment（投资） | 1 | 施粥棚前 |
| recovery（恢复） | 2 | 名医坐诊/商队休整 |
| moral（道德） | 1 | 落水书生 |
| info（信息） | 2 | 市集传闻/算命先生 |
| environment（环境） | 2 | 暴风雨至/桥梁冲毁 |
| strategy（策略） | 1 | 斗茶大赛 |
| story（剧情） | 3 | 故人重逢/偶遇诗仙/避雨茅屋 |
| exploration（探索） | 2 | 迷路岔口/神秘木匣 |
| supply（补给） | 1 | 驿站补给（每轮保底1次） |
| rare（稀有） | 2 | 黑市商人/神秘行脚僧 |

### 选项机制

- **固定效果**：银两/声望/体力/商途值固定数值变化
- **赌博🎲**：`gamble` 类型，标注胜率 (0.3~0.7)，胜败两组效果
- **范围收益**：`gains` 类型，如 `{silver:{min:150,max:200}}`
- **道具奖励**：`itemReward` 类型，直接给道具
- **道具加成**：`itemOption`，持有特定道具可用（护身符免伤、幸运符提胜率等）
- **跳转**：`jumpTo` 类型，如赌坊事件可跳转到赌坊界面

### 御街事件（10个独立池）

从「🏯 外出」进入御街触发，独立于主事件池。

| 类型 | 概率 | 数量 | 事件 |
|------|------|------|------|
| 正面 | 55% | 6 | 进贡赏赐/御前才艺/藏宝图/贵人指路/御药房/御林军 |
| 中性 | 15% | 2 | 御街避雨/皇家仪仗 |
| 负面 | 30% | 3 | 御街扒手/官员刁难/假银票 |

## 关键设计决策

- 商情持仓上限 999（防套利但允许大额操作）
- 商情买入 15% 手续费，波动范围 ×0.6~2.0
- 赌坊豹子 1:10，综合 EV 目标 0.85-0.95
- 黑市令为消耗品，每次进入消耗 1 个
- 盐场离线暂停（不产出不累积风险）
- 燃烧精血整局上限 100 点（Prestige 清零）
- 店铺变卖 40%→30%→20% 递减回收，24h 冷却
- 道具动态定价 +30% 加价，每 10 级 -4%
- 供奉解锁：全部店铺 Lv.100
