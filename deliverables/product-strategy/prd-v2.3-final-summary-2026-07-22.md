## 一、5大系统模块设计概要

### 模块1：挂机收益降低

| 机制 | 参数 | 依据 |
|------|------|------|
| 离线收益(无Prestige) | 35%/4h（v2当前50%/6h） | 数析推荐，日总产出-11.1% |
| 离线收益(Prestige后) | 50%/6h（v2当前60%/8h） | 保持Prestige激励差 |
| 操作激活窗口 | 操作后15min全效→超时70%→30min后50% | 数析"操作激活"机制 |
| 伪装模式 | 视为非活跃，产出70% | 摸鱼场景适配 |
| 商机闪现 | 在线活跃窗口内5%/结算概率×2产出30s | 竞析"金色饼干"参考，正向激励在线 |

**方案选择**：固定比例（数析推荐）优于分段递减（竞析建议），因为分段递减有效比例27.5%过激且实现复杂度高。

### 模块2：道具动态定价

| 参数 | 值 | 说明 |
|------|-----|------|
| 基础费调整 | ×1.30（比用户建议的×1.50温和） | 前期增费25-30%可接受 |
| 店铺等级折扣 | floor(shopLevel/10 - 1) × 4% | Lv.10起效，每10级多4% |
| 回平点 | Lv.55 | 基础费与折扣抵消 |
| Lv.100折扣 | -22% | 体现"投资回报" |
| 总折扣上限 | 暂不启用（v2.4+声望/Prestige折扣上线时再启用-30%cap） | 析客建议 |

**道具-店铺绑定**：11个道具绑定6种店铺（药铺道具→药铺等级折扣），10个非绑定道具不受折扣影响。

### 模块3：高风险事件

| 事件类型 | 数量 | 期望值 | 触发门槛 |
|---------|------|--------|---------|
| 体力换金钱 | 3个 | EV+1.25~1.38（正期望，鼓励翻盘） | 体力>40 |
| 体力换声望 | 3个 | EV 0.85~0.88（负期望，惩罚贪心） | 声望<门槛×0.8 |
| 阶梯式事件 | 2个 | 分3阶段，每阶段可退出 | 商途值≥200 |
| 黑市血契（稀有） | 1个 | 30体力换5000银两 | 2%概率，体力>30 |

**低体力UI警告**：体力≤30黄色⚠️ / 体力≤10红色闪烁+警告横幅。此为v2.4的P0需求（V2.3-10），是高风险事件和惩罚性倒退的共同前置依赖。

### 模块4：店铺变卖

| 参数 | 值 | 防套利措施 |
|------|-----|-----------|
| 回收率(首次) | 累计升级投入×40% | 基于投入而非购买价 |
| 回收率(二次) | ×30% | 递减折损 |
| 回收率(三次+) | ×20% | 进一步递减 |
| 等级衰减 | -0.2%/级（Lv.10起） | 高等级店铺回收率更低 |
| 冷却期 | 24h | 防频繁变卖 |
| 变卖后状态 | unlocked=false, level=0 | 锁定，需重新解锁+升级 |
| 变卖所得 | **即时全额到账**（主理人裁决，非分期释放） | 摸鱼游戏银两体量小 |

**防套利验证**：变卖同店铺重升级必然亏损52%，无套利空间。析客已分析7条套利路径，全部封堵。

### 模块5：惩罚性倒退Game Over

| 机制 | 内容 |
|------|------|
| 触发条件 | 体力归零且无化险符 |
| 惩罚内容 | 银两-50% + 全店降5级(最低Lv.1) + 体力重置20 + 精力重置10 |
| 保留内容 | 声望/商途值/道具/背包/已解锁地点 |
| 新手保护 | 游戏时间<5h：惩罚减半（银两-25%+全店降2-3级） |
| 连续安全奖励 | 连续3天未致死：下次惩罚减半 |
| 化险符 | 消耗道具免死，体力保留1点 |
| Hardcore模式 | 可选开启：体力归零=强制Prestige+声望×1.2传承点 |
| 负债催收 | 银两≤0持续3天：强制拍卖最低等级店铺还债 |
| 多系统连锁崩盘 | 盐场查封+赌坊连输+黑市查抄→银两急速消耗→负债→崩盘 |

**Non-goal修改**：原"不做体力归零=Game Over" → "不做完全清档式Game Over，但引入惩罚性倒退机制"
