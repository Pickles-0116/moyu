# 《掌柜偷闲录》背包与道具系统设计规格

**日期**：2026-07-21
**类型**：功能设计规格 / PRD 增补
**作者**：析客（需求分析师）
**状态**：待主理人审核

---

## TL;DR

- 背包与道具系统为**P1优先级**，建议在 v1.1 留存增强版本中交付核心功能，v1.2 深度版本中交付永久道具与商途值上限机制
- 新增 **9项需求**（P1五项12人日 + P2四项8人日），需求池总量从28项/79人日更新为 **37项/99人日**
- 设计了 **3大类18个具体道具**（消耗型10个 + 永久型6个 + 特殊型2个），全部古风命名
- 新增 **5个道具相关事件**，与现有事件系统无缝整合
- 关于用户提出的"商途值上限/回复商途值"：设计为**每轮商途值软上限机制** + **商途值加成道具**，同时覆盖精力上限/回复精力

## 1. 系统概述

### 1.1 定位

背包与道具系统是《掌柜偷闲录》的**战略深度层**，在核心循环（行进→事件→选择→资源变化）之上增加一层"准备与决策"维度：

```
┌─────────────────────────────────────────────────────────┐
│                    道具系统层次定位                        │
│                                                         │
│  战略深度层（新增）：背包管理 + 道具使用 + 永久升级        │
│      ↕ 为核心循环提供策略选择和资源调控手段                │
│  核心循环层（已有）：行进 → 事件 → A/B选择 → 资源变化      │
│      ↕ 依赖资源系统驱动                                   │
│  资源系统层（已有）：银两/声望/体力/商途值 + 精力          │
│      ↕ 被动产出支撑                                       │
│  经营基础层（已有）：店铺系统 + 离线收益                    │
└─────────────────────────────────────────────────────────┘
```

### 1.2 与现有系统的关系

| 现有系统 | 关系描述 |
|---------|---------|
| **四维资源系统** | 道具直接作用于体力（回复/上限）、精力（回复/上限）、商途值（加成/上限）。银两是购买道具的主要货币，声望用于解锁高级道具 |
| **精力系统** | 精力回复道具打破"5分钟恢复1点"的自然限制，给予玩家在碎片时间内额外行动机会。精力上限道具与Prestige的"+2精力上限"叠加 |
| **店铺系统** | 店铺是道具的主要购买渠道，每种店铺售卖主题相关的道具。道具系统为银两提供新的消费出口（银两沉淀问题） |
| **事件系统** | 道具可在事件中使用（C选项），部分事件奖励道具而非资源。新增5个道具相关事件 |
| **Prestige系统** | 永久型道具效果**穿透Prestige保留**（与路段解锁/传闻/成就一致），消耗型道具清零。Prestige后解锁专属高级永久道具 |

### 1.3 优先级建议：P1

**理由**：

1. **依赖关系**：道具系统依赖店铺系统（P0-07）和事件系统（P0-08），两者在MVP中已交付，P1阶段接入无阻塞
2. **核心价值**：道具系统为银两提供消费出口（解决中后期银两沉淀问题），为玩家提供更多策略选择（提升留存深度），是"明天还想来"的关键增强
3. **不过早引入**：MVP阶段需先验证核心循环"好不好玩"，道具系统作为增强层不应干扰核心验证
4. **版本归属**：核心背包+消耗品放v1.1（留存增强），永久道具+商途值上限放v1.2（深度增强）

**快捷栏设计**：
- 主界面底部新增4格快捷栏，玩家可将常用道具从背包拖入快捷栏
- 快捷栏道具可一键使用，无需打开背包面板
- 快捷栏道具与背包道具共享库存（使用快捷栏 = 使用背包中对应道具）

### 2.3 道具堆叠规则

| 道具类型 | 是否堆叠 | 堆叠上限 | 说明 |
|---------|---------|---------|------|
| 消耗型道具 | ✅ 是 | 99个/格 | 同类道具自动堆叠在同一格 |
| 特殊道具 | ✅ 是 | 10个/格 | 护身符/回城卷轴等可少量堆叠 |
| 永久型道具 | ❌ 否 | — | 购买后**立即使用**，不进入背包，直接生效 |

### 2.4 背包操作

| 操作 | 触发方式 | 说明 |
|------|---------|------|
| 查看 | 点击道具图标 | 弹出道具详情卡（名称/效果/描述/数量） |
| 使用 | 详情卡中点击「使用」 | 消耗型道具扣减数量，效果即时生效；使用条件不满足时按钮灰色禁用 |
| 装入快捷栏 | 拖拽道具至快捷栏 / 详情卡点击「设为快捷」 | 快捷栏空位优先填入，已满时提示替换 |
| 排序 | 背包面板底部排序按钮 | 按类型（回复/增益/商途/特殊）或按稀有度（普通/稀有/传说）排序 |
| 丢弃 | 详情卡中点击「丢弃」 | **普通**道具直接丢弃；**稀有及以上**需二次确认；传说道具不可丢弃 |
| 购买扩容 | 背包面板点击「扩容锦囊」 | 银两充足时即时扩容+4格，不足时灰色禁用 |

## 5. 道具使用场景与交互设计

### 5.1 使用时机矩阵

| 场景 | 可用道具类型 | 说明 |
|------|-------------|------|
| **商途进行中（行进阶段）** | 回复体力/精力类、临时增益类、商途加成类、功能类 | 行进间隙可随时打开背包使用 |

| **伪装模式中** | ❌ 不可使用 | 伪装模式下游戏暂停，道具不可用 |

### 5.4 事件C选项（使用道具）

**设计原则**：并非所有事件都有道具选项，仅在叙事逻辑合理且玩家持有相关道具时出现。

**交互流程**：


## 6. 数值平衡设计

### 6.1 道具定价基准

以现有经济系统为锚点，推导单位资源价值：

| 资源 | 单位价值（银两） | 推导依据 |
|------|----------------|---------|
| 1点体力 | ≈2.5银两 | 体力100起步，负面事件损失20-30体力≈50-75银两等价 |
| 1点精力 | ≈25银两 | 5分钟恢复1点，1精力=1次行进≈15秒核心循环≈店铺10秒产出 |
| 1点商途值 | ≈5银两 | 每步行进+5商途值≈1次行进，行进收益≈25银两 |
| 1点声望 | ≈50银两 | 声望用于Prestige，1声望=1%永久产出加成，价值较高 |

**定价原则**：
- 消耗型道具价格 = 效果价值 × 1.0~1.5（略有溢价，因为即时使用比等待恢复更灵活）
- 永久型道具价格 = 效果价值 × 15~25倍（一次性投资，长期回报）
- 事件渠道价格 = 店铺价格的 60%~80%（鼓励参与事件）

### 6.2 消耗型道具性价比验证

| 道具 | 效果价值 | 售价 | 性价比 | 评价 |
|------|---------|------|--------|------|
| 金创药 | 20体力×2.5=50 | 50 | 1.0 | 等价交换，基础保底 |
| 人参 | 50体力×2.5=125 | 150 | 0.83 | 略贵，大剂量有溢价 |
| 灵芝 | 100体力×2.5=250 | 400 | 0.63 | 偏贵，稀有度溢价 |
| 醒神汤 | 3精力×25=75 | 100 | 0.75 | 合理，即时性溢价 |
| 清心丸 | 5精力×25=125 | 200 | 0.63 | 偏贵，同上 |
| 仙露饮 | 10精力×25=250 | 500 | 0.50 | 贵，稀有+即时双重溢价 |
| 幸运符 | 难以量化 | 80 | — | 低价策略道具，门槛低 |
| 福星高照符 | 3次×50%收益提升 | 250 | 取决于单次收益 | 中高收益场景值得 |
| 急行令 | 5商途值×2×5=50额外 | 120 | 0.42 | 偏贵，但解锁路段有价值 |
| 商途锦囊 | 150商途值×1×5=750额外 | 350 | 2.14 | 超值，但一轮限用 |

**调整说明**：商途锦囊性价比偏高（2.14），但考虑到：(1)一轮只能用一个，(2)商途值本身只用于解锁路段非核心经济，(3)获取渠道仅限黑市/成就（稀有度高），可接受。若测试中发现过于强势，可将倍率从×2降为×1.5。

### 6.3 永久型道具获取节奏

| 道具 | 价格 | 预计攒够时间 | 效果持续 |
|------|------|-------------|---------|
| 聚气丹 | 1500 | 约2-3小时 | 永久（含Prestige） |
| 千年人参 | 2000 | 约3-4小时 | 永久（含Prestige） |
| 天元丹 | 4000 | 约6-8小时 | 永久（含Prestige） |
| 商路通牒 | 3000 | 约4-5小时 | 永久（含Prestige） |
| 雪山参王 | 6000 | 约10-12小时 | 永久（含Prestige） |
| 御赐商牌 | 8000 | 约13-15小时 | 永久（含Prestige） |

**节奏设计意图**：
- 玩家在v1.1阶段（首次游玩2-5小时）能买到第一个永久道具（聚气丹/千年人参），感受"投资回报"的正反馈
- 高级永久道具（雪山参王/御赐商牌）需要10+小时积累，与Prestige门槛（10-15小时）对齐，作为Prestige后的长线目标

### 6.4 与Prestige系统的关系

| 项目 | Prestige后保留 | Prestige后清零 | 说明 |
|------|---------------|---------------|------|
| 永久型道具效果 | ✅ 保留 | — | 与路段解锁/传闻/成就一致，属于永久投资 |
| 消耗型道具（背包内） | — | ✅ 清零 | 与银两/店铺等级一致，属于当轮资产 |
| 背包扩容（格数） | — | ✅ 重置为12格 | QoL升级，与店铺等级一致 |
| 快捷栏配置 | — | ✅ 清空 | 随背包清零 |
| Prestige商店解锁 | ✅ 解锁后永久 | — | 首次Prestige后永久解锁 |

**Prestige策略价值**：永久道具穿透Prestige保留，创造了"在Prestige前将银两转化为永久道具"的策略选择——是升级店铺（会被清零）还是购买永久道具（会保留）？这增加了Prestige决策的深度。

### 6.5 消耗型道具使用频率建议

| 道具类别 | 建议使用频率 | 场景 |
|---------|-------------|------|
| 体力回复类 | 2-4次/轮商途 | 体力低于50时视情况使用，避免归零结束 |
| 精力回复类 | 1-2次/天 | 关键时刻延长游玩，不建议频繁使用（破坏精力系统节奏） |
| 临时增益类 | 0-1次/轮 | 高收益事件前使用，策略性保存 |
| 商途加成类 | 0-1次/轮 | 冲刺解锁新路段时使用 |

## 7. 商途值每轮上限机制（核心设计）

### 7.1 问题背景

用户提出"提升商途值上限"和"回复商途值"，但现有设计中商途值是只增不减的进度指标。经分析，采用以下方案同时覆盖用户意图：

| 用户诉求 | 设计方案 | 说明 |
|---------|---------|------|
| 提升体力上限 | 永久型道具直接实现 | 千年人参/雪山参王 |
| 回复体力 | 消耗型道具直接实现 | 金创药/人参/灵芝 |
| 提升精力上限（用户可能误称"商途值上限"） | 永久型道具直接实现 | 聚气丹/天元丹 |
| 回复精力（用户可能误称"回复商途值"） | 消耗型道具直接实现 | 醒神汤/清心丸/仙露饮 |
| 提升商途值上限 | **每轮商途值软上限机制** | 见下文 |
| 回复/增加商途值 | **商途值加成道具** | 急行令/商途锦囊 |

### 7.2 每轮商途值软上限机制

**设计**：

- **基础上限**：150商途值/轮（= 20步×5商途值/步 + 50完成奖励，恰好为自然最大值）
- **软上限行为**：当本轮已获商途值达到上限后，后续行进的商途值获取降至 **1/步**（而非5/步），完成奖励降至 **+10**（而非+50）
- **永久道具提升上限**：
  - 商路通牒：上限+50 → 200/轮
  - 御赐商牌：上限+100 → 250/轮
  - 可叠加：同时持有则上限+150 → 300/轮
- **加成道具不受上限约束**：
  - 急行令（×3）和商途锦囊（×2）的额外商途值**不受软上限限制**，鼓励组合使用

**玩家感知设计**：
- 商途值面板显示 `🗺️ 320/500（本轮: 145/150）`
- 接近上限时进度条变色（黄→红），提示"商途值获取效率即将降低"
- 达到上限后行进日志提示"今日商途已尽，收益减少" → 鼓励购买永久道具提升上限

**数值验证**：

| 配置 | 每轮商途值（全额） | 每轮商途值（含溢出） | 解锁中原(500)所需轮次 |
|------|-------------------|---------------------|---------------------|
| 基础（上限150） | 150 | 150+20×1+10=180 | ~3轮 |
| +商路通牒（上限200） | 200 | 200+10×1+10=220 | ~2.5轮 |
| +御赐商牌（上限250） | 250 | 250+5×1+10=265 | ~2轮 |
| +双永久（上限300） | 300 | 300（几乎不溢出） | ~1.7轮 |

### 7.3 UI表现

```
商途值面板：
🗺️ 商途值  320 / 500（中原路段）
本轮进度  ██████████████░░░░  145 / 150
          ↑ 接近上限，效率即将降低
```

### 10.2 Non-goals调整

新增以下Non-goals：

- ❌ **不做道具合成/炼丹系统**（增加复杂度，不适合碎片化摸鱼场景）
- ❌ **不做道具交易/赠送**（单机无社交，无交易需求）
- ❌ **不做道具耐久度/磨损**（增加管理负担，与轻松摸鱼调性冲突）
- ❌ **不做道具图鉴收集**（与传闻碎片系统功能重叠，避免重复）

## 14. 开放问题

1. **道具平衡性调参**：商途锦囊（×2一轮）性价比偏高（2.14），是否需要在内测后根据数据调整为×1.5？—— 建议v1.1上线后观察2周数据决定
2. **背包扩容是否穿透Prestige**：当前设计为不穿透（重置为12格），但可能让玩家觉得"扩容白买了"。是否改为穿透保留？—— 建议不穿透，与店铺等级一致，Prestige本身就是"重新开始"
3. **事件C选项的UI复杂度**：C选项需要二级菜单（选择道具），是否增加操作步骤过多？—— 建议C选项直接展示最相关的1个道具（如有），点击即用，避免二级菜单
4. **道具与每日商情的交互**：每日商情影响货物价格（P1-03），道具购买是否也受商情影响？—— 建议不受影响，道具价格固定，避免双重变量增加认知负担
